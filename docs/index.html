    介绍

    mobilesdk 是由gomobile编译的跨平台NEO/ETH钱包SDK当前支持的平台有：

    • iOS
    • android

    NEO钱包支持特性

    1. 全局资产转账
    2. claim GAS
    3. WIF 导入
    4. web3 keystore support
    5. NEP5合约资产转账
    6. NEP5合约资产ICO（MintToken）

    ETH钱包支持特性

    1. 全局资产转账
    2. ERC20资产转账

    该SDK并不涉及网络部分，需要对应平台自行处理
